# Sistema-de-Control-de-Gastos-Personales
Proyecto desarrollado en Python que permite registrar y analizar gastos personales de manera simple. El sistema permite agregar gastos indicando monto y categoría, calcular el total gastado, mostrar cuánto se destinó a cada categoría y detectar automáticamente el gasto más alto y el más bajo, indicando a qué categoría pertenecen.
